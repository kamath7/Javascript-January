Prototypal Inheritance

Inheritance - One object gets access to the properties and methods of another object

Classical Inheritance - Type of Inheritance in JAVA, C# etc. 
Disadvantages - Verbose, Gorilla Jungle problem

Prototypal Inheritance - Flexible, extensible, easy to understand.
Prototype - All objects have a prototype property. It is simply a reference to another object (proto). Object look at the prototype chain for the method if it doesn't exist. 
It looks for a sequence of objects connected via the prototype chain. It's hiddenn from where the method is in the prototype chain.
